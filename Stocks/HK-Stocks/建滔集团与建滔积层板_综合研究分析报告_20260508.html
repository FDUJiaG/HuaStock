🌐 行业周期定位与驱动因素

AI驱动的CCL超级周期
山西证券（2026-05-08）："此次由AI驱动的超级周期具备很强的持续性，未来3-5年内仍将处于高速增长期"
预计CCL供需紧张格局将维持到2027年甚至更久。

核心驱动逻辑

1. 需求端：AI服务器/数据中心 → 高频高速CCL需求暴增 → 高端产品占比提升
2. 供给端：全球CCL产能扩张周期长（2-3年），短期无法快速释放 → 供需缺口持续
3. 价格端：铜价高企 + 环氧树脂涨价 + 电子布紧缺 → 成本推动+需求拉动双驱动涨价
4. 竞争格局：建滔系全球最大CCL生产商，垂直一体化成本优势在涨价周期中最为受益
5. 价格对比（弹性空间）：传统电子布~7元/米 → 第一代low DK 27元/米 → 第二代low DK 121元/米（17倍差距）
